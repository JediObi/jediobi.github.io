tomcat nio简介

Tomcat作为web服务器，
接收client的Socket,解析其中的HttPServletRequest然后传递给后台的java web，由java web解析。
接收后台java web的HttpServletResponse封装到Socket中传给client.

而Tomcat在从Socket中获取HttpServletRequest的过程中完成了高性能的nio过程。
