2. g++ commands

  • 2.1 preprocess, usually use .i as file extension for this process.

  • 2.2 compile

  • 2.3 assemble

  • 2.4 link

    g++ helloworld.cpp -o helloworld

    ~:g++ helloworld.cpp -o helloworld

4. compile and link

~:g++ helloworld.cpp

~:./a.out
